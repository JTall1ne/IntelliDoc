IntelliDoc Documentation
========================

This documentation covers the usage and development of the IntelliDoc project.

Getting Started
---------------

IntelliDoc automatically generates and updates code documentation based on repository changes. To use IntelliDoc:

#. Install dependencies: see `api/requirements.txt`.
#. Run the API server: ``uvicorn api.app:app --reload``.
#. Use the CLI commands: ``python cli/main.py --help``.

For contributors, please see the ``CONTRIBUTING.md`` file.
